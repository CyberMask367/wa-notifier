Webhook Endpoint

POST to a webhook URL to send a WhatsApp message. All JSON fields in the body are available as {{variable}} in the message template.

POST http://your-server:3000/webhook/{slug}
Content-Type: application/json

{ "status": "down", "host": "server01", "cpu": "98%" }

Template message: "🚨 Alert: {{host}} is {{status}} — CPU at {{cpu}}"

Usage Examples
curl
curl -X POST http://your-server:3000/webhook/server-alerts \
  -H "Content-Type: application/json" \
  -d '{"host":"web01","status":"down","cpu":"98%"}'
Python
import requests
requests.post(
  "http://your-server:3000/webhook/server-alerts",
  json={"host": "web01", "status": "down"}
)

Jellyfin
Send WhatsApp notifications for Jellyfin events
Setup

In Jellyfin go to Dashboard → Notifications → (your webhook plugin) and set the URL to:

http://your-server:3000/jellyfin

Enable Send All Properties in the plugin so all variables are available. The poster image is sent automatically.

Create Rule
e.g. if NotificationUsername = john → send to john's number. If status = approved → use different template.
